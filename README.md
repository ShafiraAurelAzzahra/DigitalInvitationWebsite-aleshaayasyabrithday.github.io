# Digital Invitation Website – Birthday Event

## 📌 Deskripsi
Project ini merupakan website undangan digital interaktif yang dibuat untuk acara ulang tahun. Website dirancang dengan tampilan menarik dan pengalaman pengguna yang engaging, serta dapat diakses secara online.

Project ini dikembangkan secara mandiri (solo project) sebagai implementasi pembuatan website interaktif untuk kebutuhan event digital.

---

## 🌐 Live Demo
👉 https://shafiraaurelazzahra.github.io/DigitalInvitationWebsite-aleshaayasyabrithday.github.io/

---

## 🚀 Fitur Utama
- 🎉 Tampilan undangan interaktif  
- ⏳ Countdown timer menuju acara  
- 🔊 Background music (audio)  
- 📱 Desain responsif (mobile & desktop)  
- 🎨 Animasi dan efek visual
